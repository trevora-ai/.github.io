# Residential Home Solutions Website

A modern, responsive website for Residential Home Solutions - a window, door, and siding replacement company serving Spokane Valley, WA.

## Features

- Fully responsive design (mobile, tablet, desktop)
- Modern, clean UI with professional color scheme
- Interactive contact form with validation
- Smooth scrolling navigation
- Mobile-friendly hamburger menu
- Testimonials section
- Service showcase with icons
- Contact information with business hours
- Footer with social links and quick navigation
