Avista Utilities

Window & Door Rebates

Avista customers can receive rebates for qualifying energy-efficient window and door replacements.

  • $180 per Window Contractor install - U-Factor .29 to .23
  • $225 per Window Contractor install - U-Factor .22 or below
  • $480 per Patio Door Contractor install - U-Factor .29 to .23
  • $600 per Patio Door Contractor install - U-Factor .22 or below
  • $100 per Exterior Door - ENERGY STAR® Certified

Avista Rebate Checklist

  • Completed installation by approved contractor
  • ENERGY STAR certified products
  • NFRC sticker present on windows
  • Application submitted within 90 days
